About Color Code Converter

This tool helps you convert between different color formats:

  • HEX: Six-digit color codes used in CSS (#RRGGBB)
  • RGB: Red, Green, Blue values (0-255)
  • HSL: Hue (0-360), Saturation (0-100%), Lightness (0-100%)
  • Live color preview and easy copy functionality
  • Perfect for web development and design work
